## CRUD операции с базой данных через PDO

Работа с git (предварительная работа)  
```
запустите Git Bash 
перейдите в каталог c:/xampp/htdocs
> cd c:/xampp/htdocs

Выполните Fork этого репозитория в свой аккаунт:
- Войдите в свой аккаунт на github.com и сделайте fork
на репу https://github.com/Coopteh/crud
- Снимите галочку с "Copy the main branch only" 

Настройка локального репозитория - клонируйте свой форкнутый репозиторий по SSH-ссылке
> git clone git@....
- в .gitignore основного проекта добавьте папку crud  
Перейдите в папку crud
cd crud

Создайте файлы:
- входа index.php
- .gitignore (укажите в нем папку vendor в виде исключения) 
- composer.json (настройте пространство имен Crud на папку src)

Создайте структуру MVC-проекта внутри папки /src:
- папки src/Models, src/Views, src/Controllers
- создайте автозагрузчик классов
```

Работа с базой данных
```
Запустите XAMP + Apache + mySQL, откройте phpMyAdmin
Создайте базу данных example1 и таблицу table1 в ней с 2 полями:
id (int, PK, AI), name (VARCHAR(200))
Заполните таблицу 3 записями:
(1,'Земля'), (1,'Луна'), (1,'Марс')
```

Основное задание  
используйте документацию по [PDO](https://www.php.net/manual/ru/book.pdo.php)  
видео по теме [PHP + PDO. Работа с MySQL](https://www.youtube.com/watch?v=a9l3QPMqZ1Q)
```
Создайте проект с 1 контроллером RecordController, который вызывается из точки входа index.php
внутри метода контроллера для get-запроса идет обращение:
- к модели Record для получения всех записей таблицы table1 базы данных example1
- к представлению RecordView для отображения всех полученных записей в html формате, в виде таблицы
(для отображения таблицы используйте стили Bootstrap)
```
Добейтесь, чтобы все 3 записи из базы данных отобразились на экране

Закоммитьте и запуште изменения
```
> git status
> git add .
> git status
> git commit -m "Crud структура проекта"
> git push
```
Сдайте работу - создав запрос на изменения Pull Request  
- зайдите на github и создайте Pull Request со своего аккаунта в исходный репозиторий (для аккаунта Coopteh)
